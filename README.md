# homebrew-image-gen

Homebrew tap for [image-gen](https://github.com/tolo/image_gen_cli) — a CLI that
generates images with OpenAI GPT-Image models using ChatGPT-subscription
credentials.

## Install

```bash
brew install tolo/image-gen/image-gen
```

The formula in `Formula/image-gen.rb` is rendered and pushed automatically by the
`Release Binaries` workflow in the main repo on each `v*` tag.
